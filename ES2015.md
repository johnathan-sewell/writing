
Enhanced Object Literals

http://babeljs.io/docs/learn-es2015/#enhanced-object-literals

// Shorthand property names
```JavaScript
var name = 'Jonathan';
var interests = ['ES2015', 'ES5'];

//ES5:
var me = { name: name };
me.interests = interests;

//ES2015:
var me = {name};
Object.assign(me, {interests});

// me looks like this:
// {
//     name: 'Jonathan',
//     interests: ['ES2015', 'ES5']
// }
```

// Shorthand methods
```JavaScript
//ES6
var dog = {
    bark: function() {
        console.log('woof');
    }
};
//ES2015
var dog = {
    bark() {
        console.log('woof');
    }
};
// dog.bark() logs 'woof'
```
